# jQuery 事件

jQuery 是为事件处理特别设计的。

# 什么是事件？

页面对不同访问者的响应叫做事件。

事件处理程序指的是当 HTML 中发生某些事件时所调用的方法。

实例：

  • 在元素上移动鼠标。
  • 选取单选按钮
  • 点击元素

在事件中经常使用术语 "触发"（或 "激发"）例如： "当您按下按键时触发 keypress 事件"。

常见 DOM 事件：
