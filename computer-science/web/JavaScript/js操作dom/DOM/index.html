DOM

WebAPIs保护BOM和DOM

Dom是一套操作页面元素的API，把HTML看作文档树，通过提供的API对树上的节点进行操作

DOM工作模式：先加载文档静态内容，再动态刷新，动态不影响文档静态

对页面内容刷新却不在浏览器刷新页面

  1. D

    Document 文档对象

    创建网页并加载到web，网页文档转换为文档对象

  2. O

    Object 对象

    用户定义对象(user-defined object) 程序员自创的对象

    内建对象(native object) 内建在js里的对象，如Array Math Date等

    宿主对象(host object) 浏览器提供的对象 宿主对象中最基础的对象是 window 对象。document 对象是 window 对象的一个成员属性。document 对象的主要功能是处理页面内容。

  3. M

    Model模型

    DOM把一份文档表示为一棵家谱树，家谱树本身是一种模型，与家族树相比，节点树更准确

  4. 节点

    节点是网络术语，表示网络中的一个连接点。一个网络就是由一些节点构成的集合

    DOM也是如此，文档是由节点构成的集合，此时的节点是文档树上的树枝和树叶(分支)

    4.1元素节点

    DOM的原子是元素节点(element node)

    4.2文本节点

    XHTML文档里文本节点总被包含元素节点内部。但也非所有元素节点都包含在元素节点内部

    4.3属性节点

    属性节点用来对元素做出更具体描述

    //

    hello world

    title=”a”是一个属性节点，可以被css调用

    属性总被放在起始标签里，属性节点总被包含在元素节点内。并非所有元素包含属性，但所有属性被元素包含

    DOM属性

    getElementById(元素id)

    返回一个与那个有着给定id属性值的元素节点对应对象

    只有一个参数：想获得的那个元素的id属性的值，id值放在单引或双引中

    getElementByTagName（标签名）

    返回一个对象数组，每个对象对应文档里给定标签一个元素

    只有一个参数的函数，参数是标签名

    允许把一个通配符（*）作为参数，意味着文档里每个元素都在这个函数所返回的数组里有有一席之地

    alert(document.getElementByTagName(“*”).length)查看文档总共有多少个元素节点

    getElementByClassName（类名）

    接受一个参数，参数是类名

    与TagName类似，是一个具有相同类名元素的数组

    getAttribute函数，查询属性名字

    不能被document对象调用，只能通过元素节点对象调用

    setAttribute函数，对属性节点更改，只应用元素节点

    有两个参数：属性名和属性值

    都只能与TagName方法合用，获取每个标签元素的CSS属性

    addLoadEvent函数

    只有一个参数：打算在页面加载完毕时执行的函数名

    nodeName

    总是返回一个大写字母的值

    nodeValue

    检索和设置文本节点的值

    nodeType

    共有12种可取值

    元素节点属性值是1

    属性节点是2

    文本节点是3

    childNodes[0]

    获取任何一个元素的所有子元素，是一个包含这个元素全部子元素的数组，属性返回的是一个数组

    数组包含说有类型节点

    firstChild

    访问第一个childNodes数组第一个元素

    lastChild

    访问最后一个childNodes数组第一个元素

    nextSibling

    目标元素的下一个兄弟元素即目标元素的nextSibling属性

    innerHTML

    读，写某给定元素里的HTML内容

    createElement

    创建新元素节点

    appendChild

    把创建的新元素插入到某个文档的节点树里

    createTextNode

    创建文本节点

    insertBefore

    把一个新元素插入到一个现有元素的前面

    父元素.insertBefore（新元素，插入哪个元素之前）

    元素节点的父元素必须是另一个与元素节点

    insertAfter

    参考脚本

    检索和添加信息

    对文档现有信息进行检索

    getElementById

    getElementByTagName

    getAttribute

    把信息添加到文档里去

    createEletment

    createTextNode
