HTML链接

<a href="url">文本、图片、元素</a>

超链接可以是一个字，一个词，或者一组词，也可以是一幅图像，您可以点击这些内容来跳转到新的文档或者当前文档中的某个部分

链接分类

1.外部链接（其他网页的地址）

2.内部链接（网页内部的地址）

3.空连接（#代表没有链接目标）

4.下载链接（地址是一个文件或者压缩包）

5.网页元素链接（网页中的各种元素添加的超链接）

6.锚点链接（跳转到页面位置）

href属性中属性值为#名字（id属性）

请始终将正斜杠添加到子文件夹。假如这样书写链接：href=”https://www.runoob.com/html"，就会向服务器产生两次 HTTP 请求。这是因为服务器会添加正斜杠到这个地址，然后创建一个新的请求，就像这样：href=”https://www.runoob.com/html/"

a标签中可以设置target属性，有4种可选值，分别为_blank（在新窗口打开链接）、parent（会在父窗口或者包含来超链接引用的框架的框架集中打开链接）、self（当前页面打开链接）、top（会清除所有被包含的框架，并打开链接）
