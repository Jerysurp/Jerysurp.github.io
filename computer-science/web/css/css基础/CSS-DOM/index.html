# 三位一体的网页

# 结构层 (structural layer)

网页的结构层由 HTML 或 XHTML 之类的标记语言创建。标签 (tag) 对网页内容的语义含义做出描述。标签不包含任何关于内容如何显示的信息

# 表示层 (presentation layer)

由 CSS 完成。CSS 描述页面内容如何呈现

# 行为层 (behavior layer)

负责内容如何响应事件。这是 javascript 和 DOM 的领域

# style 属性

文档中每个元素都是一个对象，每个对象又有着各种各样的属性。有一些属性告诉我们元素在节点树上的位置信息。如 parentNode、nextSibling、previousSibling、childNodes、firstChild 和 lastChild 属性告诉我们文档中各节点之间关系信息
